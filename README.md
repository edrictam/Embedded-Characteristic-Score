# Embedded Characteristic Score (ECS) — Notebooks

This folder contains the ECS experiment notebooks (MNIST, CIFAR-10, paper simulations, and chest X-ray experiments).

## Datasets

The code in the notebooks is set up to **download all relevant datasets** as needed.

- **Chest X-ray dataset size**: the chest X-ray dataset download can be **large** (in total **> 40GB**). If you run the download cells, please be patient and expect it to take a while depending on your network/disk speed.

## Outputs

- **`figures/`**: currently an **empty** directory. All output figures generated by the notebooks will be saved here.

## Paper

The relevant paper/preprint can be found at "Assessing the Distributional Fidelity of Synthetic Chest X-rays using the Embedded Characteristic Score" by Edric Tam and Barbara Engelhardt. Link is https://arxiv.org/abs/2501.00744.
